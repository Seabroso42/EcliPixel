## Como executar o programa

1 - Clone o repositório

```git clone https://github.com/Seabroso42/EcliPixel.git```

2 - Instale os requisitos
```
cd gui
pip install -r  requirements.txt
```

3 - Execute a aplicação
```
streamlit run app.py
```
